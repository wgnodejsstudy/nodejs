## Path of Exile Trade

### Usage

Repository clone
```bash
$ git clone https://github.com/wgnodejsstudy/nodejs.git
```

로컬 repository에서 아래 명령어를 실행. `localhost:3000` 페이지가 열리면서 `Hello Path of Exile`이 화면에 출력되면 성공!
```bash
$ git checkout feature/poe-trade
$ cd PathOfExile/poe-trade
$ npm install
$ npm start
```
